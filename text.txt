  → Gestion Utilisateur : si on clique là-dessus une liste déroulante contenant deux options :
      o Ajouter Utilisateur
      o Afficher Utilisateurs
  Si on clique sur l’un des deux options la fenêtre de l’ajout ou d’affichage s’ouvre
  → Gestion Personne : si on clique là-dessus une liste déroulante contenant deux options :

      o Afficher apprenants

      o Afficher professeurs

      o Afficher responsables

      o Ajouter matière
      o Afficher matières
  → Gestion des salles :
      o Ajouter salle
      o Afficher salles
  → Gestion des séances :
      o Ajouter séance
      o Afficher séances
  → Un bouton de filtrage pour la méthode afficherSeanceProfesseur
  → Un tableau pour lister toutes les séances, ce tableau contiendra

de la séance
